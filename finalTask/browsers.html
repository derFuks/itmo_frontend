Кроссбраузерность и выводы

Довольно ленивая проверка работы страниц в разных браузерах и выводы о кроссбраузерности.

Функция Chrome (актуальный) Firefox (актуальный) Safari Проклятый IE
Разметка HTML5 OK OK OK Частично
Новые типы инпутов (date, email, tel, range, number) OK OK OK Отображаются как обычный text, но не ломают форму
WebStorage (localStorage / sessionStorage) OK OK OK ОК
SVG OK OK OK PNG показал
Видео (MP4/WebM) OK OK OK OK
CSS3 (flex, transitions, border-radius, media queries) OK OK OK Есть отличия в оформлении

Выводы

  • Сайт остается читаемым в проверенных браузерах.
  • Новые элементы форм HTML5 в старых браузерах как текстовые поля отобразились.
  • WebStorage работает при из-за проверок try/catch
  • SVG и видео дублируются PNG и WEBM на всякий случай.
