Для кого этот сайт нужен?

Это демо проект по курсу разработка Web-приложений на HTML5 CSS3 и JavaScript.
На примере травматологов и неврологов показываю:

  • верстку шапки и меню с использованием header и <nav>.
  • Формы отправки заявки на запись с элементами: дата, e-mail, телефон и т.д.
  • Хранение данных о записи с помощью WebStorage. В БД для диплома переделаю запись.
  • SVG-логотип и видео через тег video.
  • Гораздо больше элементов CSS3, чем нужно по заданию: flex, transitions, border-radius и адаптив через медиа-запросы.

Заявка на запись к врачу онлайн

Форма ниже записывает данные в sessionStorage. При успешной отправке должно появиться сообщение об успехе, иначе смотри консоль.

Я не делал обработки задаваемых пользователем значений в формах. Осознанно. На дипломе сделаю.

При ошибках валидации сообщения выводятся в console.error. Сами данные не отправляются ни на сервер, ни в файл. Только записываются в sessionStorage.
